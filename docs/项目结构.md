## 分层说明

- `frontend/`：用户界面层，负责页面展示、交互、表格排序和调用后端 API。
- `backend/routes/`：API 接口层，负责 URL、请求参数、登录态校验和 JSON 响应。
- `backend/services/`：业务逻辑层，封装登录查询、图书查询、学生创建、管理员创建等业务。
- `backend/models/`：SQLAlchemy 模型层，描述学生、管理员、图书、馆藏副本、借阅、预约、违期等数据对象。
- `backend/mysql_cli.py`：数据库访问工具层，封装 MySQL 命令行调用和 SQL 转义。
- `database/`：数据库定义层，包含建表、初始数据、触发器、函数和存储过程。
- `docs/`：课程项目文档层，包含需求分析、接口说明、用户说明和结构说明。

## 运行模式

本项目只保留 Vue + Flask 前后端分离模式：

- Flask 后端 API：`http://127.0.0.1:8000`
- Vue 前端页面：`http://127.0.0.1:5173`

Vue 前端通过 `frontend/vite.config.js` 将 `/api` 请求代理到 Flask 后端。
